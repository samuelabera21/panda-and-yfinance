# Python OOP Beginner Guide

This project helps beginners learn Python OOP step by step, from core concepts to simple real-world apps.

## What you will learn

- Class and Object
- Encapsulation
- Abstraction
- Polymorphism
- Inheritance (single, multilevel, hierarchical)
- How OOP is used in small practical projects

## Project structure

- `main.py` → runs all demos
- `basic_oop/` → core OOP concepts
  - `class_and_object.py`
  - `encapsulation.py`
  - `abstraction.py`
  - `polymorphism.py`
  - `demo.py`
- `inheritance/` → inheritance examples
  - `single_inheritance.py`
  - `multilevel_inheritance.py`
  - `hierarchical_inheritance.py`
  - `demo.py`
- `real_app/` → grading system mini project
  - `grading_system.py`
  - `demo.py`
- `library_system/` → library management mini project
  - `library_management.py`
  - `demo.py`
- `calculator_app/` → calculator mini project
  - `calculator.py`
  - `demo.py`

## Real-world mini projects

### 1) Grading System
- Uses classes like `Person`, `Student`, and `GradeBook`
- Shows inheritance and encapsulation
- Includes methods to add students, add scores, and generate reports

### 2) Library System
- Uses classes like `Book`, `Member`, and `Library`
- Shows object interaction and state handling
- Includes methods to issue and return books

### 3) Calculator App
- Uses a `Calculator` class
- Shows class methods for arithmetic operations (`+`, `-`, `*`, `/`, `**`, `%`)
- Includes operation history, clear history, and simple input validation
- Handles edge cases like divide-by-zero safely

## How to run

1. Open terminal in this folder.
2. Run:

```bash
python main.py
```

If `python` does not work, run:

```bash
python3 main.py
```

## Suggested learning path

1. Start with `basic_oop/`
2. Continue with `inheritance/`
3. Study `real_app/grading_system.py`
4. Study `library_system/library_management.py`
5. Study `calculator_app/calculator.py`

Each folder has a small demo file, and `main.py` runs everything together.
